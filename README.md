# DEVvsBUGS

Назва гри: Dev vs BUGS
Тема: GAME DEV

Історія:
П’ятниця, 17:04. Ти вже відчуваєш смак вихідних і плануєш запустити Software Inc., як раптом у Mattermost «прилітає» таск від тімліда: термінова фіча на понеділок. Перед очима тисячі рядків легасі-коду без жодного коментаря... Свідомість туманиться, і ти прокидаєшся всередині власного проекту. Навколо — нескінченна темрява коду, а з усіх боків лізуть баги.

Мета: Пережити цю фічу.
Core Loop:
    Боротьба: Автоматично "відстрілюй" баги (фіксуй код).

    Прогресія: Збирай досвід за вбиті баги, підвищуй рівень та обирай пасивні чи активні апгрейди статів.

    Виживання: Стеж за шкалою "Вигорання" — якщо вона заповниться, я не хочу казати, що буде коли вона заповниться...

Керування:

    Стрілочки: Рух героя.

    X: Ривок (Dodge).

Mood-board:

    Pixel Art: Чистий, зрозумілий 2D-стиль із чітким фідбеком.

    Інтерфейс: Мінімалістичний UI, що відображає стан ментального здоров'я розробника.
